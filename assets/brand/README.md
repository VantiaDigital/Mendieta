Todo derivado del feed real de `@pasteleriamendieta` (paleta y tipografía
extraídas pixel-exacto, no a ojo).

Canva siguiendo la guía de abajo.

## Cómo cargarlo en Canva (paso a paso)

Entrá: Inicio → **Más** (···) → **Marca** → Brand Kit **Mendieta**.

### 1) Logos
`Logos` → **Subir** → arrastrá los 5 archivos de `logos/`.
- Marcá `mendieta-perro-tinta.png` como **logo principal**.
- Los demás quedan como variantes para distintos fondos.

### 2) Colores
`Colores` → **+ Añadir paleta** → en cada color **+ Añadir nuevo** y pegá:
```
FBF4C6   Crema      (fondo principal)
77231B   Bordó      (titulares)
533118   Cacao      (subtítulos)
936D4C   Caramelo   (iconos, líneas)
EDC77D   Mostaza    (badges, bordes)
1B1613   Tinta      (cuerpo, captions)
```
Renombrá cada color con su nombre para reconocerlo al diseñar.

### 3) Fuentes
`Fonts` → seteá cada rol (todas están en Canva, buscalas por nombre):
```
Encabezado  →  Rye               (titulares con filete: BINGO, MEDIALUNAS)
Subtítulo   →  Playfair Display  (Favoritos, ARGENTINOS VIVIENDO EN)
Cuerpo      →  Montserrat        (párrafos, captions)
```

### 4) Gráficos (Canva Pro)
`Gráficos` → **Subir** → arrastrá los archivos de `graphics/`.
Quedan disponibles para arrastrar a cualquier diseño.

### 5) Voz de marca (Canva Pro)
`Voz de marca` → pegá el resumen de tono del `manual-de-marca.md`
(sección 6: voseo argentino, cálido, nostálgico, sin venta agresiva).

## Reglas rápidas (detalle en guidelines/manual-de-marca.md)

- **Fondo**: crema casi siempre.
- **Titular**: bordó + Rye/Playfair. Máx 3 palabras en Rye.
- **Máximo 3 colores y 2 fuentes por pieza.**
- **Logo**: con espacio de respeto, nunca deformado ni recoloreado.
- **Voz**: voseo argentino, cálido, "vení / llevate / extrañás".
- **Foto**: real, luz cálida, producto protagonista.

---

## Pendiente / a confirmar con el cliente

1. **Logo vectorial**: las variantes salen de un screenshot de baja resolución.
   Para print/gran formato, pedir el SVG/AI original al diseñador.
2. **Fuente decorativa exacta**: Rye es la más parecida en Canva a la serif con
   filete del feed. Si el cliente confirma el nombre real, reemplazar.
3. **Fotos de producto por SKU**: para web y posts, falta banco de fotos
   individuales (medialuna, alfajor, empanada, etc.) sobre fondo limpio.

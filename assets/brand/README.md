### Cómo cargar en Canva

1. Entrá a tu Brand Kit en Canva (Pro): https://www.canva.com/brand
2. **Colores de marca** → **Añadir nuevo**
3. Pegá cada HEX de arriba uno por uno
4. Nombralos exactamente como acá: `Crema`, `Bordó`, `Cacao`,
   `Caramelo`, `Mostaza`, `Tinta` (para que los reconozcas rápido al diseñar)

### Cuándo usar cada color

- **Crema**: fondo de todo. Si dudás, el fondo va en crema.
- **Bordó**: titulares grandes ("MENDIETA", "BINGO", "NUESTRAS MEDIALUNAS").
  Nunca lo uses para cuerpo de texto largo (se cansa la vista).
- **Cacao**: subtítulos display debajo del titular bordó
  ("de Mendieta", "Año Nuevo"). También para sellos.
- **Caramelo**: iconos finos, líneas decorativas, contornos de ilustraciones.
- **Mostaza**: bordes de avatar/highlights, badges destacados,
  ocasionalmente pills de info.
- **Tinta**: textos largos (captions). Casi nunca en pieza diseñada;
  más útil para chat / captions de IG.

### Combinaciones recomendadas

```
Fondo Crema + Titular Bordó + Subtítulo Cacao + Detalle Caramelo
Fondo Crema + Texto Tinta sobre Bordó destacado
Foto oscura + Texto BLANCO (caso especial: 'MIRÁ DE CERCA', 'TORTA ROGEL')
```

### Lo que NO se hace

- ❌ Texto crema sobre fondo crema (sin contraste)
- ❌ Bordó sobre Cacao (compiten, se ven sucios)
- ❌ Mostaza como fondo grande (cansa)
- ❌ Más de 3 colores en una misma pieza
